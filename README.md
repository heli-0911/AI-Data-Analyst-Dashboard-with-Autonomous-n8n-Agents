## 🤖 AI Agents (n8n Automation)

Integrated n8n workflows to build AI-powered automation agents that enhance the dashboard capabilities.

### 🔹 What it does:
- Automates data analysis requests
- Triggers workflows based on user queries
- Processes insights and sends responses back to dashboard
- Enables intelligent decision-making

### 🔹 Example Flow:
User Query → Flask API → n8n Agent → Data Processing → Response → UI

### 🔹 Tools Used:
- n8n (Workflow Automation)
- Webhooks
- AI Logic Handling
